My Job

  1. User Observation : Conduct a needs analysis from the school, librarians, and students to understand the features and functions required in the system.
  2. Use Case Diagram Design : Create a diagram of the interaction between users (students, teachers, librarians) and the system to map the main functions, such as book search, borrowing, and data management.
  3. UI/UX Design : Design the app's interface with a focus on ease of navigation, student-friendly aesthetics, and efficient functionality. Create wireframes and prototypes to ensure the design fits the user's needs.
  4. Frontend Development (Mobile) : Implement UI/UX design into code using frontend technologies such as HTML, CSS, and JavaScript.Create responsive apps so they can be accessed well across a variety of devices (desktop, tablet, and smartphone).
