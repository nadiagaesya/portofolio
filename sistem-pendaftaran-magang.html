Deskripsi Project

The Internship Registration System Project aims to simplify the registration process and manage intern data digitally. As a UI/UX Designer, I designed an intuitive and user-friendly interface, ensuring that the registration flow was easy for prospective attendees and admins to understand. As a Frontend Developer, I implemented the design into a responsive and functional application using modern technology, ensuring accessibility across multiple devices as well as seamless integration with the addition of a backend system for efficient data management.
