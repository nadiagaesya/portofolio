Digital Library System Website Frontend

The Digital Library System is designed to facilitate the management of book collections and student access to online library services. As a UI/UX Designer, I design user-friendly and efficient interfaces for book searching, borrowing, and library data management. As a Frontend Developer, I implement these designs into responsive web applications, ensuring features run optimally across multiple devices to support a comfortable digital experience.

My Focus

  • In this project, I made observations of the client to understand the system requirements and features needed.
  • Create use case diagrams to visually map the flow of interactions between users and the system.
  • Design an intuitive and functional interface to make it easier for prospective interns and admins to use the system. Create wireframes and prototypes independently.
  • Implement UI/UX designs into responsive and functional code using technologies such as HTML, CSS. Optimize app performance for fast loading times. Additionally, I leverage version control tools like Git for efficient collaboration with the rest of the development team.
